# Cox Dash — Constitution

## Non-Negotiable Principles
- Single HTML file, no build tools, no dependencies, runs in any browser
- No backend, no server, no database
- Playable by anyone in under 10 seconds with zero setup
- Working and fun beats perfect and broken
- Every decision must serve the demo moment

## Project Identity
- Name: Cox Dash
- Type: Endless runner browser game
- Theme: Corporate office, Cox Automotive step challenge
- Tone: Funny, competitive, personal to the room

## Hard Boundaries
- No mobile touch controls
- Persist scores via localStorage only, no backend ever
- No external assets or image files, all visuals in code
- No multiplayer networking
- One life, game over on hit
