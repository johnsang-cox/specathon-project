# Cox Dash

A browser-based endless runner built for the Cox Automotive Step Challenge 2026. Run as far as you can without getting caught by your managers.

## How to Play

Open `index.html` in a browser. No server or install needed.

| Control | Action |
|---|---|
| `SPACE` / Up Arrow | Jump |
| Down Arrow | Duck |
| Left then Right (quickly) | Juke Alex Taylor |

## The Managers

Managers swap in progressively as your step count climbs. Each one is faster than the last.

| Manager | Alias | Appears at |
|---|---|---|
| Nick | Iron Man | Start |
| Nikki | Sugar Cat Mom | 100 steps |
| Sam | Oui Oui Baguette | 200 steps |
| Andrew | Wise Sage of Beer | 300 steps |
| Alex Taylor | Aura King / Final Boss | 400 steps |

Alex Taylor can be juked with a quick left then right input. All others are unavoidable.

## Tech

Single-file HTML5 Canvas game. No frameworks, no build step.
